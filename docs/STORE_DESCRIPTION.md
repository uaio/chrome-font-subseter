# Chrome Web Store 描述信息

## 简短描述（最多80字符）
字体子集化工具 - 快速创建包含所需字符的小字体文件，优化网页加载速度

## 详细描述

字体子集化工具是一款帮助Web开发者和设计师优化字体文件的Chrome扩展。通过提取仅需要的字符，大幅减少字体文件大小，提升网页加载性能。

🚀 **主要功能：**
• 支持TTF、OTF、WOFF、WOFF2等主流字体格式
• 灵活选择需要保留的字符
• 内置常用字符集快捷按钮
• 实时预览字体效果
• 一键生成并下载字体子集
• 显示压缩率和文件大小对比

💡 **使用场景：**
• 网页性能优化
• 移动端字体优化
• 减少不必要的字符加载
• 创建多语言字体子集
• 图标字体精简

🎯 **适合人群：**
Web开发者、UI/UX设计师、前端工程师、性能优化专家

📝 **使用方法：**
1. 点击扩展图标
2. 上传或拖拽字体文件
3. 选择需要保留的字符
4. 生成并下载字体子集

⚡ **立即优化您的网站字体加载速度！**
