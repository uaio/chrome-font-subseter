# 安装指南

## 方法一：离线安装（推荐）

### Windows 用户

1. **下载插件包**
   - 下载 `chrome-font-subseter-v1.0.0.zip`
   - 解压到任意文件夹

2. **打开Chrome扩展管理页**
   - 打开Chrome浏览器
   - 地址栏输入 `chrome://extensions/`
   - 或点击右上角菜单 -> 更多工具 -> 扩展程序

3. **启用开发者模式**
   - 页面右上角打开"开发者模式"开关

4. **安装插件**
   - 点击"加载已解压的扩展程序"
   - 选择解压后的文件夹
   - 安装成功！

### Mac 用户

1. **下载插件包**
   - 下载 `chrome-font-subseter-v1.0.0.zip`
   - 双击解压

2. **打开Chrome扩展管理页**
   - Chrome菜单 -> 偏好设置 -> 扩展程序
   - 或地址栏输入 `chrome://extensions/`

3. **启用开发者模式**
   - 打开"开发者模式"开关

4. **安装插件**
   - 点击"加载已解压的扩展程序"
   - 选择解压后的文件夹
   - 完成！

## 方法二：GitHub直接安装

1. 访问项目GitHub页面
2. 点击绿色的 "Code" 按钮
3. 选择 "Download ZIP"
4. 解压下载的文件
5. 按照方法一的步骤安装

## 方法三：源码安装（开发者）

```bash
# 克隆仓库
git clone https://github.com/uaio/chrome-font-subseter.git

# 进入目录
cd chrome-font-subseter

# 在Chrome中加载该目录
```

## 常见问题

### Q: 提示"扩展程序无效"怎么办？
A: 确保解压后的文件夹包含 manifest.json 文件

### Q: 更新插件
A:
1. 下载新版本
2. 在Chrome扩展管理页点击"更新"
3. 或删除旧版本后重新安装

### Q: Win10提示安全警告
A: 点击"更多信息" -> "仍要运行"
