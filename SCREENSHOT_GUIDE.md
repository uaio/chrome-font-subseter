# 截图指南

## 截图要求

### 尺寸规格
- **推荐尺寸**：1280x800（16:10）或 640x400（16:10）
- **格式**：PNG或JPG
- **数量**：至少1张，最多5张
- **文件大小**：每张不超过1MB

### 截图内容建议

#### 1. 主界面展示（必选）
- 上传区域清晰可见
- 标题"字体子集化工具"醒目
- 显示拖拽提示文字

#### 2. 字符输入界面
- 快捷字符按钮（数字、字母、汉字）
- 字符输入文本框
- 字符统计信息

#### 3. 字体预览功能
- 上传字体后的预览效果
- 显示"预览"标签
- 示例文字展示

#### 4. 生成结果展示
- 原始大小、子集大小、压缩率
- 进度条或成功提示
- 清晰的数据对比

#### 5. 下载选项
- 格式选择（WOFF2、WOFF、TTF、OTF）
- 下载按钮
- 格式说明文字

### 拍摄技巧

1. **去除浏览器边框**
   - 使用Chrome截图工具或系统截图工具
   - 确保只显示扩展内容

2. **使用清晰字体**
   - 截图要清晰可读
   - 避免模糊或像素化

3. **适当的内容填充**
   - 上传区域可以显示一个示例字体文件名
   - 字符输入框可以填入示例字符
   - 预览区域显示实际效果

4. **保持界面整洁**
   - 关闭不必要的标签页
   - 使用简洁的背景

### 截图工具推荐

#### Mac用户
1. **系统截图**：Cmd+Shift+4（区域截图）
2. **Chrome开发者工具**
   - 右键 -> 检查
   - Device toolbar (Ctrl+Shift+M)
   - Screenshot按钮

#### Windows用户
1. **系统截图**：Win+Shift+S
2. **Snip & Sketch**应用

### 示例文件名建议
```
screenshot-01-main.png
screenshot-02-input.png
screenshot-03-preview.png
screenshot-04-result.png
screenshot-05-download.png
```

### 上传到开发者控制台
1. 进入Chrome Developer Dashboard
2. 在"商店列表"页面找到"上传截图"
3. 拖拽或选择准备好的截图文件
4. 确保所有截图都符合要求

### 注意事项
- 不要包含个人或敏感信息
- 不要包含其他扩展或浏览器界面
- 确保截图内容真实反映扩展功能
- 保持界面美观和专业
