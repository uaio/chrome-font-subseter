# 图标文件说明

这个目录包含Chrome插件所需的图标文件。

## 图标尺寸

- **icon16.png** - 16x16 像素 - 用于扩展列表
- **icon32.png** - 32x32 像素 - 用于Windows等系统
- **icon48.png** - 48x48 像素 - 用于扩展管理页面
- **icon128.png** - 128x128 像素 - 用于Chrome Web Store

## 创建图标

可以使用以下方式创建图标：

1. 使用图标设计工具（如 Figma、Sketch 等）
2. 使用在线图标生成器
3. 从图标库获取（确保有合适的许可证）

## 图标设计建议

- 主题：字体子集化、字体优化、压缩等
- 风格：简洁、现代、易于识别
- 颜色：使用品牌色或醒目的颜色组合

## 临时解决方案

目前可以使用任何临时的PNG图标文件，之后可以替换为设计的最终版本。
